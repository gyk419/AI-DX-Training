4. Analysis Scenarios

This dataset is designed to support the following statistical analysis scenarios:

# Scenario Analysis Type Expected Finding
1 Before/After Balance Change Paired t-test Significant improvement after exercise
2 Age Group Balance Comparison ANOVA 20s-30s show best balance
3 Gender Pressure Difference Independent t-test Males show slightly higher pressure
4 Exercise Type Effect ANOVA + Post-hoc Jump exercise most effective
5 BMI-Balance Correlation Correlation Analysis Higher BMI correlates with lower balance
6 Injury History Impact Independent t-test Injured users show lower balance
7 Seasonal Exercise Effect ANOVA Summer most effective
8 Time of Day Pattern ANOVA Afternoon optimal
9 Device Type Comparison ANOVA No significant difference (control)
10 Frequency-Grade Relationship Chi-square Test Higher frequency → higher A-grade ratio
11 Age × Gender Interaction Two-way ANOVA Interaction effect present
12 Time-Series Improvement Time Series Analysis Stabilization after 4 weeks

5. Usage Example

import pandas as pd

# Load dataset
df = pd.read_csv('foot_pressure_demo.csv')

# Basic exploration
print(df.info())
print(df.describe())

# Scenario 1: Paired t-test (Before/After)
from scipy.stats import ttest_rel

df_wide = df.pivot(index='user_id', columns='condition', values='balance_ratio')
t_stat, p_value = ttest_rel(
    abs(df_wide['before'] - 1.0),
    abs(df_wide['after'] - 1.0)
)
print(f"Paired t-test: t={t_stat:.4f}, p={p_value:.6f}")

# Scenario 4: ANOVA (Exercise Type)
from scipy.stats import f_oneway

groups = [group['balance_ratio'] for name, group in df.groupby('exercise_type')]
f_stat, p_value = f_oneway(*groups)
print(f"ANOVA (Exercise Type): F={f_stat:.4f}, p={p_value:.6f}")
